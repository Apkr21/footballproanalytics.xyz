GitHub Pages Deployment Guide

Step 1: GitHub Repository

  1. Create GitHub account at github.com
  2. Click "New repository"
  3. Name it "footballpro-analytics"
  4. Make it Public (free)
  5. Check "Add README file"
  6. Click "Create repository"

Step 2: Upload Files

  1. Save this page as "index.html"
  2. In GitHub, click "uploading an existing file"
  3. Drag and drop your index.html file
  4. Add commit message: "Initial website"
  5. Click "Commit changes"

Step 3: Enable GitHub Pages

  1. Go to repository Settings tab
  2. Scroll to "Pages" section
  3. Source: "Deploy from a branch"
  4. Branch: "main" (or "master")
  5. Folder: "/ (root)"
  6. Click "Save"

Step 4: Custom Domain

  1. In Pages settings, enter: footballproanalytics.xyz
  2. GitHub will create CNAME file
  3. In Namecheap DNS settings:
  4. Add CNAME record: www → username.github.io
  5. Add A records for apex domain (see below)
  6. Wait 10-30 minutes for DNS propagation

DNS Records for Namecheap:

A Records (for apex domain):
@ → 185.199.108.153
@ → 185.199.109.153
@ → 185.199.110.153
@ → 185.199.111.153
CNAME Record:
www → yourusername.github.io
Replace "yourusername" with your GitHub username

Success Checklist:

  • ✓ Repository created and files uploaded
  • ✓ GitHub Pages enabled
  • ✓ Custom domain configured
  • ✓ DNS records added in Namecheap
  • ✓ SSL certificate auto-generated (may take up to 24 hours)
  • ✓ Site accessible at footballproanalytics.xyz

PayPal Integration Setup

Replace PayPal Button IDs

You need to replace the placeholder button IDs in this HTML file with your actual PayPal button IDs:

Daily Tips (€4.99): DAILY_TIPS_BUTTON_ID
5 Tips Package (€9.99): PACKAGE_BUTTON_ID
Pro Monthly (€19.99): PRO_MONTHLY_BUTTON_ID
Premium (€49.99): PREMIUM_BUTTON_ID

How to Find Your Button IDs:

  1. Log into your PayPal account
  2. Go to Tools → All Tools → PayPal Buttons
  3. Find your created button
  4. Click "View Code" or "Get Code"
  5. Look for: hosted_button_id" value="XXXXXXXX"
  6. Copy the XXXXXXXX part (that's your button ID)

How to Replace in HTML:

  1. Open this HTML file in text editor
  2. Use Ctrl+F to find "DAILY_TIPS_BUTTON_ID"
  3. Replace with your actual button ID
  4. Repeat for all 4 button IDs
  5. Save the file
  6. Upload updated file to GitHub
